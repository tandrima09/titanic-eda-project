# 🚢 Titanic Survival - Exploratory Data Analysis

## 📌 Objective
This project explores the Titanic dataset to understand which factors 
(gender, passenger class, age, family status, embarkation port) influenced 
a passenger's chance of survival.

## 🛠️ Tools Used
- Python
- Pandas
- Matplotlib & Seaborn
- Google Colab

## 🔍 Key Insights
- `adult_male` showed a strong negative correlation with survival
- Passengers traveling alone had only a 30.4% survival rate
- Cherbourg passengers survived more — likely linked to passenger class, not the port itself
- Traveling with family positively impacted survival chances

## 📊 Process
1. Data Loading & Setup
2. Exploration & Understanding
3. Missing Value Handling
4. Visualization
5. Insight Generation
6. Final Report
